# 森林生态学刷题

这是一个从本地 Word 题库和术语表整理出的静态刷题网页。

当前内容：
- 211 道森林生态学选择题
- 190 条生态学专业术语

功能：
- 随机练习：每轮随机抽取 20 道选择题
- 错题本：自动记录做错题目和累计做错次数
- 英译汉：每轮随机抽取 10 条术语，只做英文到中文
- 汉译英：每轮随机抽取 10 条术语，只做中文到英文
- 提交后即时判分，并用绿色答案条显示正确答案
- 页面底部也有提交、换题和清空按钮，答完不用回到顶部

打开 `index.html` 即可使用。发布到 GitHub Pages 后，也可以通过 Pages URL 在线访问。
